
# Todo App React

Todo app created using create-react-app. Styling done only though CSS.




## How to run

Download dependencies and run:
```bash
  cd todo-app-react  
  npm install
  npm start
```


## Features

- Add todos
- Edit todos. Click a todo to edit it.
- Mark a todo as done/undone or delete/undelete it.
